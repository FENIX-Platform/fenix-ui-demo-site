FENIX Platform

WEB BASED STATISTICAL SYSTEM

Overview

An internet-based and scalable system able to meet different user requirements and adaptable to support global, regional and national level projects and initiatives. The main goal is to contribute to creating a global data sharing network for agriculture and food security by establishing a scalable system integrating many different data sources.

FENIX adheres to the Open Data initiative and is designed to avail data and technologies to anyone as “public good” using free, open-source software and providing data at no cost. We aim at facilitating collaboration between countries and organizations to build a wide data exchange and analysis network.

Main Feature

  • 1 Based on Open Source software; no licensing constraints and full redistribution rights
  • 2 Using a Service-Oriented Architecture (SOA)
  • 3 Extensible: ability to plug-in new functionalities and applications (widgets)
  • 4 Designed to facilitate data sharing through web services (APIs)
  • 5 Using diversified data collection/upload methods (online data entry; uploads from csv/xls files; mobile apps)
  • 6 Able to handle databases, geospatial data (as remote sensing, GIS layers, etc.) and text
  • 7 An advanced analytical application with the embedded “R” statistical package
